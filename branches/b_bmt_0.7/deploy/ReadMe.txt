Typical directory structure:
BMT:
 |
 |-> libs
 |
 |-> src
 |
 |-> BMT_vXX.jar
 |
 |-> (DLL/SO files)
 
Description:
1. libs: 			all jar libraries needed for the class-path (Required- libs_vXX.zip)
2. src: 			source code (Optional- src_v_XX.zip)
3. BMT_vXX.jar: 	runnable jar file
4. DLL/SO files: 	link libraries used by the program (Required- link_libs_vXX.zip)
